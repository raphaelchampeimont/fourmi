La version SDL est bien meilleure mais elle nécessite
la librarie SDL.

Ce programme marche sous tout système UNIX.

VERSION PPM
Pour compiler la version avec rendu PPM faites :
  make
Et lancez la avec :
  ./fourmi

VERSION SDL
Pour compiler la version avec rendu en temps réel SDL faites :
  make sfourmi
Et pour la lancer :
  ./sfourmi
